== Description ==

The NesiaPress theme can be used for Personal Blogs, Portfolios, or Business Websites. With responsive, SEO, lightweight features, and woocomerce support, you can also build an online store website using this WordPress theme. NesiaPress layout can change to center, full-width, wide-block or using sidebar.

Learn more and check out the NesiaPress feature on https://nesiapress.com

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in 'NesiaPress' in the search form and press the 'Enter' key on your keyboard.
3. Click on the 'Install' and 'Activate' button to use NesiaPress theme right away.

== NesiaPress Child Theme ==

You can custom NesiaPress theme layout using child theme:

1. Download the child theme from our Github page - https://github.com/kukuhnovaputra/nesiapress-child
2. More information on creating child theme - https://codex.wordpress.org/Child_Themes
