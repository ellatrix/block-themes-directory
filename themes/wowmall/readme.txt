== Description ==

Wowmall is a multi-purpose block theme designed for WooCommerce. Itâ€™s been built entirely around the Full Site Editing features available in WordPress 5.9, and thanks to the layout features of Full Site Editing, you can easily adapt it for any type of website.

Wowmall includes multiple block styles and block patterns including site headers, site footers and page layouts.
