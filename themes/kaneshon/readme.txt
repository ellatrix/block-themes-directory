== Description ==

Kaneshon is a clean and elegant WordPress theme designed specifically for the new Full Site Editing experience. Through the new Site Editor you can not only customize site colors and typography but also the entire layout including the header, footer and page and post templates. Kaneshon locally hosts 10 popular Google Font families to give you flexible typography while remaining GPDR compliant. It also comes with three custom block patterns for hero sections, post queries and social engagement.
