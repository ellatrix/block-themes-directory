== Description ==

Renovater is a modern and responsive style theme template for WordPress fullsite editing and fully compatible with Gutenverse plugin. Renovater created especially for Construction, Building Companies and Freelancer website for companies or individuals that need a modern landing page for their business. The templates includes both core version and Gutenverse plugin version, also has core and Gutenverse block patterns ready so you can start mix and match your template parts as you desire. The templates is built ready so you don't need to build it from scratch. We want to make your experience using WordPress fullsite editor more convenient.
