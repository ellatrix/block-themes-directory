Experimental Full-Site Editing Theme.

== Description ==
Pacer is a modern block-based WordPress theme, especially made for blog, news, and magazine. It is fully compatible with all the new features of WordPress 5.9. Quick responsiveness, various block styles, and patterns have eased it to adapt for any type of website. You can build an entire website overnight using the Pacer WordPress theme. Also, you can customize global fonts and colors anytime with a single click. This theme allows you to take the potential of full site editing.
