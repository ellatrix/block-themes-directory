== Description ==

Hypebiz is a modern, responsive, and clean-styled theme template for WordPress fullsite editing and is fully compatible with the Gutenverse plugin. Hypebiz is perfect for Business, Consulting Services, Consultants, Finance, Enterprises, Financial Advisors, Digital Agencies, and any type of business that provides consulting services. The templates is built ready so you don't need to build it from scratch. We want to make your experience using WordPress fullsite editor more convenient.
