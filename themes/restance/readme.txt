== Description ==

Restance is a simple, responsive, and elegant looking theme template for WordPress fullsite editing and fully compatible with Gutenverse plugin. Restance's templates is built with layout/design that suit for Real Estate sites and Furniture Design. The templates includes both core version and Gutenverse plugin version, also has core and Gutenverse block patterns ready so you can start mix and match your template parts as you desire. The templates is built ready so you don't need to build it from scratch. We want to make your experience using WordPress fullsite editor more convenient.
