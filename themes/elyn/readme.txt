## Description

Elyn is a Full Site Editing first WordPress theme based on a minimal design style. Pre-defined block patterns are included.
Global style options makes Elyn easy to customise. If your looking to a lightning fast theme built for the future of WordPress look no further than Elyn. 

Demo: https://elyn.unisonthemes.com/
