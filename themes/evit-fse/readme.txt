== Description ==
EVIT FSE is a block-based WooCommerce-compatible theme with dedicated support,flexible customization options,font and color control,and pre-made demos for various industries. A real demo is available at the provided link: https://demos.eaglevisionit.com/evit-fse

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
