== Description ==

Trouvelot is a simple full site editing (FSE) block theme built with portfolios and blogs in mind, inspired by The Trouvelot Astronomical Drawings published in 1882, a collection of 15 exquisite pastel works created by the French artist.

This theme is beta software, and is not meant for use on a production site.
