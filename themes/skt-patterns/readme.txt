== Description ==

SKT Patterns is a modern responsive interior design and architect related WordPress theme which can be edited using block editor. It also makes use of full site editing (FSE) features. It also includes patterns. All aspects including colors, and typography are editable using the block editor. This theme can also be used for portfolio, website design, construction related websites.
